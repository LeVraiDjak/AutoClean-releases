# AutoClean — Téléchargements officiels

**AutoClean** nettoie et organise intelligemment vos fichiers : repérez les fichiers anciens et volumineux, éliminez les doublons et libérez de l'espace en toute sécurité. Par défaut, les fichiers sont envoyés à la corbeille — rien n'est supprimé sans votre accord.

Ce dépôt héberge uniquement les versions publiées de l'application.

## 📥 Installation

Téléchargez la dernière version depuis la page **[Releases](../../releases/latest)** :

| Système | Fichier | Installation |
|---|---|---|
| Windows 10/11 | `AutoClean-windows.zip` | Dézippez, puis lancez `AutoClean\AutoClean.exe` |
| Linux | `AutoClean-linux.tar.gz` | `tar -xzf AutoClean-linux.tar.gz` puis `./AutoClean.bin` |
| macOS | `AutoClean-macos.app.zip` | Dézippez, puis clic droit sur `AutoClean.app` → Ouvrir (première fois) |

> **Windows** peut afficher un avertissement SmartScreen tant que l'application n'est pas signée : cliquez sur « Informations complémentaires » → « Exécuter quand même ».
> **macOS** : l'application n'étant pas encore notarisée, utilisez clic droit → Ouvrir au premier lancement.

## 🔄 Mises à jour

AutoClean vérifie automatiquement les nouvelles versions au démarrage (désactivable dans les Paramètres). Les pré-versions sont proposées uniquement si vous activez « Inclure les pré-versions ».
